## 🚨 The Problem

Municipal systems across India are plagued by deep inefficiencies that erode public trust and waste taxpayer money:

| Issue | Impact |
|---|---|
| 👻 **Ghost Workers & Proxy Attendance** | Fraudulent wage claims with no service delivery |
| 📡 **Zero Real-Time Field Visibility** | Admins can't verify where workers are at any moment |
| 🔍 **Unverifiable Task Completion** | No proof system — citizens bear the consequences |
| 📋 **Paper-Based Records** | Easy to forge, impossible to audit, costly to store |

> *Traditional governance has no way to confirm a pothole was actually filled, a drain was actually cleared, or a worker was actually present.*

## 💡 What is CivicTrack?

<div align="center">

### *"The operating system for accountable city governance."*

</div>

**CivicTrack** is an Android-based municipal workforce management platform that transforms field operations into a fully **digital, trackable, and auditable ecosystem**.

From garbage collection to road repairs — every task is **verified**, **timestamped**, and **geotagged**.

It creates a three-way trust bridge:

```
🏠 Citizens  ←→  📊 Admins  ←→  👷 Field Workers
    ↑                ↑                ↑
  Transparency    Control         Accountability
```

## ✨ Features

### 📍 GPS Check-In / Check-Out
Workers mark attendance only when physically within a designated **geofenced zone** — eliminating proxy attendance entirely.

### 📸 Geotagged Photo Evidence
Every completed task is documented with a **watermarked, geotagged photo** — irrefutable proof tied to time and location.

### 🔒 Geofencing Attendance
Custom geofences per zone ensure attendance can't be marked from home, cafes, or anywhere outside the work area.

### 📊 Real-Time Admin Dashboard
Command-center visibility with live worker locations, task statuses, complaint queues, and performance KPIs — all in one screen.

### 🤖 ML Anomaly Detection
Firebase ML Kit flags suspicious patterns — repeated check-ins from the same GPS coordinates, unusually fast task completions, and duplicate photo submissions.

### 🏙️ Citizen Complaint Portal
Residents can report civic issues with photo evidence, track resolution progress live, and rate service quality — closing the loop between citizens and governance.
